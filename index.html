ripple is an open source decentralized payment network.

Decentralized

a decentralized peer-to-peer network -- which is why no one owns or controls it.

Virtual currency

a virtual currency, so you can shop or send money within ripple more efficiently.

Open Source

open source, so you can build on top of ripple and create new features or services.

People powered

a people-powered payment platform, which lets you become your own paypal or buy and sell currency.
